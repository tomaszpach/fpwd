## Setup guide
1. Clone or download Repository.
2. Go to main directory and open console / terminal
3. Follow steps listed below:

### Yarn

1. yarn
2. yarn start

### NPM
1. npm install
2. npm start

Now you are able to see App in your browser on http://localhost:3000
